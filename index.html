    THE UNSOLVED PROBLEM

    Floating-Point Inference Is Fundamentally Non-Deterministic

    Every major AI lab ships models that produce different numerical outputs on different hardware. Not approximately different. Actually different. CPU vs GPU. Driver A vs Driver B. Batch of 1 vs batch of 32. The same model, the same weights, different bits.

    This is not a bug. IEEE-754 floating-point arithmetic is non-associative. Operation reordering, FMA fusion, subnormal flushing, signed-zero behavior, NaN propagation — all of these produce legitimate but divergent results. No existing standard resolves this for inference-scale logit streams.

    CONSEQUENCE Model audit, reproducibility, on-chain commitment, and cross-hardware verification are impossible without a canonical layer. PCMP is that layer.

    TECHNICAL DEPTH

    Inside the Canonical Pipeline

    SPCMP · Preprocessing Canonicalization pseudocode 
    // Step 1: SPCMP Preprocessing — CVD enforcement
function spcmp_preprocess(u: uint32) → uint32 {
  // NaN canonicalization (any payload → canonical qNaN)
  if is_nan(u):
    return 0x7FC00000
  // Signed zero elimination
  if u == 0x80000000:
    return 0x00000000
  // All other values preserved exactly
  return u
}

// Step 2: PCMP canonical ordering mapping
function pcmp_map(u: uint32) → uint32 {
  if (u & 0x80000000):
    return ~u          // negative: flip all bits
  else:
    return u | 0x80000000  // positive: set sign bit
}

// Step 3: Sort → predict → compress → Merkle
function pcmp_canonicalize(stream: float32[]) → MerkleRoot {
  canonical = stream.map(spcmp_preprocess)
  mapped    = canonical.map(pcmp_map)
  sorted    = mapped.sort()               // unsigned compare
  predicted = delta_encode(sorted)
  compressed = lz_compress(predicted)
  return merkle_tree(compressed, chunk=4096).root
}
    ORDERING GUARANTEE Unsigned integer comparison of mapped values. Total order. No stability dependency.
    CVD PROPERTY Many-to-one. Intentionally non-invertible outside domain. Loss is normative, not a bug.
    HASH DOMAIN Computed over compressed predictive data. Not raw floats. Not canonical floats. Exact byte stream only.
    FAIL SEMANTICS Fail-closed. Any step violation causes immediate hard failure. No partial verification.
    VERSION POLICY Version 1 frozen forever. Canonicalization rules immutable. Modifications require new profile identifier.
    ADVERSARIAL BOUND No adversary on SPCMP artifacts can recover NaN payloads, distinguish ±0, or create equivalent artifacts with different hashes.
